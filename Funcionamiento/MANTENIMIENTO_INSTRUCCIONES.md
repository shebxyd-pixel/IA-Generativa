# Archivos de mantenimiento y pruebas

En esta carpeta se encuentran todos los archivos dedicados a las pruebas de mantenimiento y a la comprobación del funcionamiento general del sistema.

## Descripción

Estos archivos están pensados para verificar que todo funcione correctamente, detectar posibles fallos y validar la ejecución de los distintos componentes.

## Contenido

- Documentos de prueba
- Scripts de verificación
- Archivos auxiliares que solo son necesarios para confirmar el correcto funcionamiento

## Uso recomendado

- Si eres desarrollador o tienes experiencia técnica, puedes revisar y ejecutar estos archivos para verificar el comportamiento del proyecto.
- Si no estás familiarizado con la configuración interna del sistema, no es necesario que interactúes con ellos.

## Nota para usuarios no técnicos

En caso de que no seas un usuario con experiencia o que no te interese modificar la IA, puedes ignorar completamente esta carpeta. No es necesario abrir, editar ni ejecutar ninguno de sus archivos para utilizar el proyecto en su forma habitual.

## Objetivo

El propósito de esta carpeta es facilitar el mantenimiento y las pruebas internas sin interferir con la operación normal del proyecto. Está separada de los archivos principales para que solo el personal técnico la use cuando sea necesario.
